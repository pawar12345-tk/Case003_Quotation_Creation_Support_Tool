# 売上管理ポータル v1.5

## 修正内容
- 月別_グループ計の「項目」にグループ名が表示される問題を再修正。
  - 内部集計キーを文字列分解しない構造に変更。
  - グループ列は `group`、項目列は `item` として保持。
- 読込完了後に自動的に隠す対象を変更。
  - 隠す対象：PJ名／PJコード検索などの検索条件パネル、読込状況などのサマリカードパネル
  - 隠さない対象：作業フォルダ選択、手動読込、条件クリア、CSV出力の操作パネル
- 検索条件パネル・サマリカードパネルはマウスオーバーで再表示。
- 月別_グループ計は上期／下期／通期のラジオボタンで表示範囲を切替。
