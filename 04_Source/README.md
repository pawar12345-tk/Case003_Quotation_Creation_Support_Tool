# 売上管理ポータル v2.0

## 変更内容
- `sales_manage_portal.html` を単体HTMLのまま全面再構成。
- CSS / HTML / JavaScript を読みやすい構造に整理。
- 今回・前回・予算の読込、検索条件、KPI、タブ、一覧を業務ツール向けの落ち着いたUIに再設計。
- 金額列は見出し前方の全角スペースを除去し、同名見出しが複数ある場合は合算して集計。
- 表示件数、金額単位、年度・チーム・区分・ランク・取引先・PJ期間フィルタを追加整理。
- 月別グループ計は上期／下期／通期を切替可能。
- CSV出力は現在表示中のタブ・条件・表示件数に合わせて出力。
- 選択した作業フォルダをブラウザ内の IndexedDB に保存し、次回起動時に自動読込。
- 表以外の操作・条件・集計パネルは上部に折りたたみ、マウスオーバーまたはフォーカスで表示。

## 利用方法
- `sales_manage_portal.html` をブラウザで開く。
- `作業フォルダ選択` から、配下に `今回見通し` / `前回見通し` / `予算` フォルダを持つ作業フォルダを選択する。
- 一度選択した作業フォルダはブラウザ内に保存され、次回以降は許可が残っていれば自動で読み込まれる。
- 操作・検索条件・タブ切替は画面上部の細いバーにマウスを乗せて表示する。
- ブラウザがフォルダ選択に対応していない場合は、`手動読込` から3つのCSVを指定する。
